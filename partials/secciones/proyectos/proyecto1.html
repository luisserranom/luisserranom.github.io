Portafolio Web

En este proyecto de portafolio web, utilicé JavaScript, HTML, CSS (Grid y Flexbox) y Bootstrap 5. Está alojado en GitHub Pages, utilizando Git y GitHub para el control de versiones. Implementé partials con funciones JavaScript para un código más ordenado y conceptos básicos de SEO para mejorar la visibilidad. También optimicé el rendimiento con un sprite de imágenes.
